# Ch6 Heapsort

- sorts $$n$$ elements in $$O(nlog(n))$$
- in place
	ps. _in place_ : only used $O(1)$ additional memory to store data
- heat definition
- heat property: $$ \mbox{Parent[PARENT(i)] } \leq \mbox{ A[i]}$$
